  1. Die Einstellungen sind in die Zwischenablage kopiert.
  2. GitHub öffnet sich (mit deinem Login). Klick ins Textfeld, alles markieren (Strg+A) und einfügen (Strg+V).

Gruppe:

  1. Pi mit dem Raspberry Pi Imager flashen: Pi OS (64-bit, Desktop), Hostname + Benutzer/Passwort, SSH aktivieren, WLAN/Zeitzone.
  2. Pi starten, ein Terminal öffnen (oder per SSH verbinden).
  3. Player-Namen vergeben und den einen Befehl ausführen (Passwort eingeben, wenn gefragt):

Danach am Pi: sudo reboot. Der Bildschirm zeigt dann automatisch diese Gruppe.
